# E-Commerce Shopping Cart (Java)

A basic e-commerce shopping cart system in Java.

## Features
- Add products to cart
- View cart contents
- Calculate total price
- Demonstrates OOP principles

## How to Run

1️⃣ Compile:

```bash
javac Product.java ShoppingCart.java Runner.java
```

2️⃣ Run:

```bash
java Runner
```

Tested with Java 11+
